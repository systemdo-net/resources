## 📖 项目介绍

SystemDo 资源中心 — 精选的企业软件开发模板、工具集、白皮书与品牌资产库。为团队、客户和开发者社区提供高质量的参考资源。

汇集了我们在 10+ 年、1000+ 项目中积累的实战经验与可复用资源。

## 📂 资源分类

| 类别 | 内容 |
|------|------|
| **📋 项目模板** | 需求文档模板、方案建议书、架构设计图、部署检查清单 |
| **🛠️ 工具集合** | 开发实用工具、环境配置模板、Docker Compose 模板、CI/CD 流水线示例 |
| **📄 技术白皮书** | 企业软件架构、AI 落地实践、数字化转型等专题白皮书 |
| **🎨 品牌资产** | 品牌标识、UI 组件库、设计系统资源 |
| **📊 项目案例** | 脱敏项目案例，含架构概览与实现效果 |

## 🎯 适用场景

- **启动新的企业软件项目** — 使用模板加速需求梳理和方案规划
- **技术选型评估** — 参考白皮书和架构决策记录
- **搭建部署流水线** — 直接复用经过生产验证的 Docker 和 CI/CD 配置
- **统一品牌体验** — 使用设计系统组件和品牌规范
